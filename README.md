# ♟ Стратег КИОР — Шахматная платформа ЦИМ ЗПМ

## Что это?

Современная веб-платформа для игры в шахматы, визуально и концептуально интегрированная с проектом **ЦИМ ЗПМ** (Цифровой инструмент мониторинга для Заместителя Премьер-министра) на базе платформы **Smart Data Ukimet**.

## Для кого?

Для команды **НАО КИОР**, **МКИ РК** и **АО «НИТ»** — как демонстрация того, что продукт может выходить за рамки утилитарного инструмента и становиться частью бренда проекта.

## Почему это ценно?

Шахматы — стратегическая игра, идеально отражающая философию ЦИМ ЗПМ:  
**«От реактивного управления — к проактивному»**.

Каждый элемент интерфейса отсылает к реальной архитектуре проекта:

| Элемент игры | Аналог в ЦИМ ЗПМ |
|---|---|
| Движок ИИ «Система SDU» | Платформа НПИИ / H200 (496 GPU) |
| Оценочная шкала (eval bar) | Индекс социальной напряжённости (ИСН) |
| Сигнальная полоса (🟢🟡🔴) | EWS-алерты системы раннего предупреждения |
| «КИОР-интерпретация» | Аналитический слой НАО КИОР (271 исследование) |
| Режимы «Дебют / Миттельшпиль / Эндшпиль» | Стандартный / Оперативный / Кризисный режимы |
| Уровни сложности: Стажёр → Зам. Премьера | Ролевая модель доступа ЦИМ ЗПМ |

## Технический стек

- **HTML5 + CSS3 + Vanilla JavaScript** — нулевые зависимости
- **Алгоритм**: Minimax с альфа-бета отсечением (глубина до 4)
- **Движок**: полные правила шахмат (рокировка, взятие на проходе, превращение пешки, шах/мат/пат)

## Функционал

- Игра против ИИ (4 уровня сложности) или 2 игрока локально
- Контроль времени (3 / 10 / 30 минут или без ограничений)
- Оценочная шкала позиции в реальном времени
- История ходов в нотации SAN
- Аналитика: лучший ход, фаза игры, КИОР-интерпретация
- Сигнальная полоса в стиле EWS-алертов
- Определение дебюта (Сицилианская, Испанская, Ферзевый гамбит и др.)
- Захваченные фигуры и счёт материала
- Переворот доски, откат хода

## Запуск

```bash
# Просто открыть в браузере:
open index.html

# Или через локальный сервер:
python3 -m http.server 3000
# Затем: http://localhost:3000
```
